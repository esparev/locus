## Instalacion

Instalar el projecto con npm

```bash
  npm i
```

Correr los estilos de TailwindCSS

```bash
  npx tailwindcss -i ./src/styles/input.css -o ./src/styles/tailwind.css
```

Compilar el proyecto

```bash
  npm start
```
